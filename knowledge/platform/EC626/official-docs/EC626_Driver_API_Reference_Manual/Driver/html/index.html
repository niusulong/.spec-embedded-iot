EC626-Driver  Version 1.0
Peripheral Driver Interfaces
Overview

EC616-Driver implements peripheral drivers where Communication drivers(USART,SPI,I2C) are CMSIS-Driver specification compliant.

The following Driver API groups are defined:

  • XIC: Interrupt controller interface driver.
  • DMA: DMA(Direct Memory Access) interface driver.
  • PAD: Pinmux interface driver.
  • GPIO: General Purpose I/O interface driver.
  • TIMER: Timer interface driver.
  • WDT: WDT(Watchdog Timer) interface driver.
  • ADC: Analog-to-Digital converter interface driver.
  • USART: Universal Synchronous and Asynchronous Receiver/Transmitter interface driver.
  • I2C: Multi-master Serial Single-Ended Bus interface driver.
  • SPI: Serial Peripheral Interface Bus driver.
  • CLOCK: Clock Management interface driver.
  • SLPMAN: Sleep Manager interface driver.
