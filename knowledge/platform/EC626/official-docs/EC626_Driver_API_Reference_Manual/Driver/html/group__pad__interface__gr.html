Description

Driver API for PAD.

The PAD module provides support for signal muxing and eletrical properties control.

Pad mux configuration

uint32_t __pad1__: 1

Enable pull-up

uint32_t pullDownEnable: 1

Enable pull-down

uint32_t pullSelect: 1

Pull select, external or internal control

uint32_t inputForceDisable: 1

Force to disable input or not, if true, input path is cut off, otherwise, it's controlled by inputControl bit
