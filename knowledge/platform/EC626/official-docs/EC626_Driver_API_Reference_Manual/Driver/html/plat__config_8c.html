Note
when the order of struct plat_config_fs_t and plat_config_raw_flash_t field has changed, for example, from 1,2,3 to 1,3,2, update version to refresh flash, in other cases(add more fields or remove some fields from struct plat_config_fs_t), it's not a must to update
