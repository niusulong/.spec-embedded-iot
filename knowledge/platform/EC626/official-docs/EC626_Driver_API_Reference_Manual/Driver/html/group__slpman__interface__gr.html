◆ slpManVoteSetDozeEnable()

void slpManVoteSetDozeEnable ( bool  en)

void slpManVoteSetDozeEnable(bool en)

allow doze state or not temporary in application layer. There is also a doze switch in at+ecpcfg="wfiMode",0. Set wifMode=0 to allow doze, when wifmode=1 do not enter doze. And wfiMode configure is saved in flash. slpManVoteSetDozeEnable api only write a variable in ram to disable doze temporary. Because it does not write flash, it is valid in only active/idle/doze/sleep1. When wakeup from sleep2/hibernate, doze is configured by the wfiMode flag. And user can call slpManVoteSetDozeEnable to reconfigure doze. The variable behind slpManVoteSetDozeEnable is true by default, which means allow to enter doze.

Parameters
entrue, allow enter doze false, do not enter doze state
