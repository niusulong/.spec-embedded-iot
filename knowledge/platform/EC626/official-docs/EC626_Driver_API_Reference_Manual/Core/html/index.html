Overview

EC616-Core (Cortex-M3) implements CMSIS-Core(M) Standard which gives the user direct access to the peripheral registers. In detail it defines:

  • Interrupt Number Definition provides interrupt numbers (IRQn) for all exceptions and interrupts of the device.
  • Processor and Core Peripherals configuration reflects the features of the device.
  • Device Peripheral Access provides definitions of all device peripherals. It contains all data structures and the address mapping for device-specific peripherals.
