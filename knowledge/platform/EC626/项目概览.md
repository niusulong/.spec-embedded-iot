## 4. 构建系统分析

| 维度 | 详情 |
|------|------|
| **主构建系统** | GNU Make |
| **构建入口** | `PLAT/Makefile` → 委托给 `project/{TARGET}/apps/{PROJECT}/GCC/Makefile` |
| **编译配置** | `nwy_common.mk` (通用), `nwy_project.mk` (项目级), 各变体 `nwy_project.mk` |
| **支持工具链** | GCC (arm-none-eabi) + ARMCC (Keil) 双链支持 |
| **构建脚本** | `GccBuild_ec626.bat`, `GccBuild_ec626e.bat`, `build_nwy.bat` |
| **Kconfig** | 局部使用 (NWY_FRAMEWORK 内) |
| **预编译库** | `prebuild/FW/`, `prebuild/PS/`, `prebuild/PLAT/` (固件/协议栈/平台) |
| **构建变量** | `TARGET` (ec626_0h00), `PROJECT` (at_command, bootloader 等), `V` (verbose) |

**构建流程**：
```
build_nwy.bat / GccBuild_ec626.bat
  → PLAT/Makefile (TARGET=ec626_0h00, PROJECT=at_command)
    → project/ec626_0h00/apps/at_command/GCC/Makefile
      → 包含 nwy_common.mk, nwy_project.mk, driver/*.mk, middleware/*.mk
        → gcc 输出到 gccout/
```

**特性宏控**（从 `nwy_project.mk` 提取）：

| 宏定义 | 功能 |
|--------|------|
| `FEATURE_NWY` | Neoway 框架总开关 |
| `TYPE_EC626` | EC626 芯片类型 |
| `FEATURE_NWY_AT` | AT 命令功能 |
| `FEATURE_NWY_BLE_ENABLE` | BLE 蓝牙功能 |
| `FEATURE_NWY_AT_NET_BZ` | 网络 AT 命令 |
| `FEATURE_NWY_AT_CTLWM2M_AUTO_UPDATE` | CT LwM2M 自动更新 |
| `FEATURE_NWY_AT_ONENET_AUTO_UPDATE` | OneNET 自动更新 |
| `FEATURE_NWY_NYSM_SUPPORT` | Neoway SM 支持 |
| `FEATURE_NWY_AT_UART_BAUD_ADAPT` | UART 波特率自适应 |
| `FEATURE_PS_TCPIP_SKTSOCK_AT_ENABLE` | TCP/IP Socket AT 命令 |
| `FEATURE_NWY_FS_AT_ENABLE` | 文件系统 AT 命令 |
