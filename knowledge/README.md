# 中央知识库

跨项目持久化的知识文档，按芯片平台组织，独立于任何代码仓库。

## 目录结构

```
~/.agents/knowledge/
├── platform/
│   └── {平台名}/                    # 如 EC626、ASR1603
│       ├── 项目概览.md              # 由 spec-project-overview 生成
│       └── code-summary/           # 由 spec-code-summary 生成
│           └── {模块名}/
│               └── 代码总结.md
└── README.md                        # 本文件
```

## 与项目 .spec 目录的关系

| 位置 | 用途 | 特点 |
|------|------|------|
| `~/.agents/knowledge/platform/{平台}/` | 跨项目通用知识（代码总结、项目概览） | 持久化，不随项目删除 |
| 项目 `.spec/` | 项目级工作数据（bug、需求、日志） | 跟项目走 |

## 使用规则

- 平台目录和子目录由各技能按需创建，不预创建
- 平台名由用户提供或从项目路径推断
- 文档间使用相对路径引用
- 同名文档更新时覆盖旧版本
