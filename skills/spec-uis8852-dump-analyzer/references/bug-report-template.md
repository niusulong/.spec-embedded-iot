## 编写要点

1. **区分"已验证"与"推断"**：每个结论标注证据来源（脚本输出/反汇编/源码/推测）
2. **中断身份用 `g_osIrqNo` 权威确认**，不要只靠栈扫描猜
3. **崩溃时刻函数用帧感知回溯 + 反汇编 call site 验证**，不要轻信启发式扫描
4. **dlmalloc assert 先看堆使用率**：耗尽也会触发 539（dlmallocPrint 遍历撞异常）
5. **malloc trace ring 末尾非崩溃 alloc**：崩溃时刻调用者看栈，不看 ring
6. **复现路径必须给**：证据不足时给已知条件 + 推测，标"待验证"
